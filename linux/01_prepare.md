# Übung 1: Vorbereiten der Umgebung

1. Dateien vom USB Stick in einen lokalen Ordner kopieren
2. Winrar installieren um die TGZ Datei zu entpacken
3. TGZ Datei entpacken
4. VMWare Workstation installieren

3. Starten der VM
